## ✨ Overview

BLACKBOX is not a typical gallery app.

It was designed as a small private internet world where memories, anonymous thoughts, hidden vault entries and images can live in a more immersive way.

Think:

🕯 quiet archive  
🌌 cinematic atmosphere  
📦 private storage  
💭 anonymous fragments  
🔒 hidden spaces  

---

## 🚀 Features

### 📸 Memory Archive
Store and upload memories with images and captions.

✔ Dynamic image gallery  
✔ Responsive layout  
✔ Hover animations  
✔ Masonry style archive  

---

### 💭 Anonymous Fragments

Users can leave private messages and thoughts.

Examples:

```text
Some things deserve a quieter place.

2:13 a.m. felt different.

Not every memory asks to be loud.
```

---

### 🔒 Vault

Password-protected private section.

Use it for:

- hidden notes
- private memories
- restricted content
- secret archives

---

### ⚙ Admin Dashboard

Control center for managing content.

Includes:

- image uploads
- content review
- archive controls
- private management tools

---

### 🎨 Cinematic UI

BLACKBOX uses:

- glassmorphism
- layered blur
- ambient effects
- floating visuals
- smooth transitions
- premium interactions

## ⚡ Installation

Clone repository:

```bash
git clone https://github.com/Moksh-dev/blackbox-dynamic.git
```

Open folder:

```bash
cd blackbox-dynamic
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
python app.py
```

Open:

```txt
http://127.0.0.1:5000
```
